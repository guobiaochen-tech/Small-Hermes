# 🏛️ Small Hermes 小白安装教程

> **写给完全不懂编程的你** —— 一步一步教你把 AI 聊天软件装到自己电脑上。

---

## 📦 总共需要装 3 样东西

| 序号 | 软件 | 干啥用的 | 难不难 |
|------|------|----------|--------|
| ① | Node.js | 让代码能运行 | ⭐ 下载安装就行 |
| ② | Ollama | 让 AI 模型在你电脑上跑 | ⭐ 下载安装 + 等它下载完 |
| ③ | Small Hermes | 聊天界面本身 | ⭐⭐ 敲几个命令 |

**全部装完大概需要**：30分钟～1小时（主要看网速，因为要下载 18GB 的模型）

---

## 第一步：装 Node.js（5分钟）

1. 打开浏览器，访问 **https://nodejs.org**
2. 页面中间有个绿色的 **「LTS」** 按钮，点它下载
3. 下载完打开安装包，一路点 **「下一步」** 直到完成
4. **重启电脑**（重要！不重启后面可能识别不到）

> ✅ **验证：** 打开「终端」App（在「启动台」->「其他」里），输入 `node -v`，如果能显示版本号（比如 `v24.15.0`），说明装好了。

---

## 第二步：装 Ollama + 下载模型（20～40分钟）

### 2.1 安装 Ollama

1. 访问 **https://ollama.com**
2. 点 **Download**，下载 macOS 版
3. 打开下载的文件，把 Ollama 图标拖进「应用程序」文件夹
4. 打开 Ollama（第一次会提示你安装命令行工具，点确定）

### 2.2 下载 AI 模型

1. 打开「终端」App
2. 输入以下命令（可以直接复制粘贴，然后按回车）：

```bash
ollama pull gemma4:26b
```

3. **等着**。这个要下载大概 18GB 的数据，屏幕会一直滚动显示进度
4. **什么时候好？** 等终端重新出现 `$` 符号，或者显示 `success` 就说明下完了

> 💡 **小提示：** 下载时间取决于你的网速，WiFi 快的十几分钟，慢的可能要一小时。中途可以最小化窗口干别的事。

---

## 第三步：装 Small Hermes（10分钟）

### 3.1 下载代码

打开「终端」，**逐条**输入以下命令（每输完一条按一次回车）：

```bash
cd ~/Desktop
```

```bash
git clone https://github.com/guobiaochen-tech/Small-Hermes.git
```

```bash
cd Small-Hermes
```

这时候桌面上会出现一个叫 `Small-Hermes` 的文件夹。

### 3.2 安装依赖

继续在终端里输入：

```bash
npm install
```

屏幕会滚动一些字，**等它停住不动**（出现 `$` 符号）就完成了。

### 3.3 启动！

```bash
npm run dev
```

终端会显示类似这样的内容：

```
[Vite] Dev server running at: http://localhost:5173
[Server] Running on port 3000
```

### 3.4 开始聊天

1. 打开浏览器
2. 在地址栏输入 **http://localhost:5173**
3. 按回车 → 看到聊天界面了！
4. 输入你的问题，点发送

**恭喜！你已经可以在自己的电脑上和 AI 聊天了 🎉**

## ⚠️ 常见问题

### Q: 终端报 `command not found: node`
→ Node.js 没装好，或者没重启电脑。重启后再试。

### Q: 终端报 `command not found: git`
→ 你的 Mac 没装 Git。去 https://git-scm.com 下载安装，或者直接用浏览器打开 GitHub 链接，点绿色的「Code」->「Download ZIP」，解压到桌面。

### Q: 浏览器打开 `localhost:5173` 显示不了
→ 确认终端还在运行 `npm run dev`，不要关掉它。最小化就行。

### Q: 聊天回复很慢
→ 正常。这个模型大概 18GB，在你自己的电脑上跑，每次回答需要几秒钟。云端的 ChatGPT 快是因为人家用几千张显卡。

### Q: 我想在飞书上用
→ 这个稍微复杂一点，需要注册飞书开发者账号、创建应用、配置 Webhook。如果你需要，我另外写教程。

## 🔧 每次怎么用

以后想用的时候：

1. 打开 **Ollama**（确保它在运行，菜单栏有小图标）
2. 打开「终端」
3. 输入：

```bash
cd ~/Desktop/Small-Hermes
npm run dev
```

4. 浏览器打开 **http://localhost:5173**
5. 开始聊天

用完直接关掉终端窗口就行。
